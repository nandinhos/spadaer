# Technology Stack - SPADAER

## Core Backend
- **Language:** PHP ^8.2 (Running on PHP 8.4 container)
- **Framework:** Laravel ^12.0
- **Database:** MariaDB/MySQL (via Docker/Laravel Sail)
- **Authentication/Authorization:** Laravel Breeze / Spatie Laravel Permission

## Frontend
- **Framework:** Livewire ^4.1
- **Styling:** Tailwind CSS
- **Interactivity:** Alpine.js v3
- **Templating:** Blade

## Data & Reports
- **PDF Generation:** Barryvdh Laravel DOMPDF ^3.1
- **Excel/Spreadsheets:** Maatwebsite Excel ^3.1

## Infrastructure & Dev Tools
- **Environment:** Laravel Sail (Docker)
- **Package Management:** Composer (PHP), NPM (JS)
- **Code Style:** Laravel Pint
